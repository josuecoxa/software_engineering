# TechFlow Solutions - Sistema de Gerenciamento de Tarefas

## 📌 Objetivo do Projeto
Este projeto foi desenvolvido para a **TechFlow Solutions**, uma empresa contratada por uma startup de logística. O objetivo é fornecer um sistema de gerenciamento de tarefas ágil que permita acompanhar o fluxo de trabalho em tempo real, priorizar tarefas e monitorar o desempenho da equipe.

## 🛠️ Escopo
O sistema contempla um CRUD básico (Create, Read, Update, Delete) de tarefas, permitindo o registro, acompanhamento e atualização dos status das atividades desenvolvidas pela equipe de logística.

## 🔄 Metodologia Adotada
O projeto foi gerenciado utilizando **Metodologias Ágeis**, com forte base no **Kanban**. 
O fluxo de trabalho foi dividido e organizado na aba *Projects* deste repositório, garantindo transparência através das colunas: `To Do`, `In Progress` e `Done`.

## 💻 Tecnologias Utilizadas
- **Linguagem:** Java
- **IDE:** IntelliJ IDEA
- **Testes Automatizados:** JUnit / Mockito
- **CI/CD:** GitHub Actions (Pipeline para execução de testes automatizados)

## 🚀 Como executar o sistema
1. Clone o repositório: `git clone https://github.com/josuecoxa/TechFlow-Task-Manager.git`
2. Abra o projeto na sua IDE (IntelliJ IDEA).
3. Aguarde o download das dependências (Maven/Gradle).
4. Execute a classe `Main` (ou `Application`, se usar Spring Boot).
