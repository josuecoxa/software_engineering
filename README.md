# TechFlow Task Manager

Sistema de gerenciamento de tarefas (CRUD) desenvolvido em Java como parte do projeto de Engenharia de Software da Faculdade.

## 🚀 Funcionalidades
* Criar Tarefas com título, descrição e prioridade (Alta, Média, Baixa)
* Listar Tarefas cadastradas no console
* Atualizar o Status das tarefas (To Do, In Progress, Done)
* Excluir Tarefas do sistema

## 🧪 Tecnologias e Qualidade
* **Java 17** (Desenvolvimento do Core)
* **Maven** (Gerenciamento de dependências)
* **JUnit 5** (Testes unitários automatizados)
* **GitHub Actions** (Integração Contínua - Pipeline de CI)
